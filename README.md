Как задеплоить (очень кратко)

1) GitHub:
- Создай новый репозиторий (public).
- Загрузи туда содержимое папки C:\1t\render-dialect

2) Render:
- New -> Web Service -> выбери этот репозиторий.
- Environment: Docker (или Blueprint, если Render предложит).
- После деплоя получишь URL вида https://xxxxx.onrender.com

3) GitHub Pages:
- Settings -> Pages -> Deploy from branch
- Folder: /docs
- Открой GitHub Pages URL, вставь Render URL, загрузь аудио.

Ограничения:
- Бесплатный Render может засыпать.
- Очень длинные аудио лучше не грузить.
